Tech Stack

Frontend

  • Vue 3.4 · Composition API
  • Vue Router 4.3 · history mode
  • Axios 1.6
  • Vite 5.2 · dev server
  • State composables (no Pinia)
  • i18n custom · en / ja
  • Charts custom SVG

Backend

  • Python 3.x · uv
  • FastAPI REST + OpenAPI
  • Uvicorn ASGI server
  • Pydantic v2 · validation
  • Persistence in-memory JSON

Tooling & Tests

  • pytest backend API tests
  • httpx FastAPI TestClient
  • pytest-cov coverage
  • Scripts start.sh / stop.sh
  • Frontend tests none
